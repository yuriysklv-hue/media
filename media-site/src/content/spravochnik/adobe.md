## О компании

Adobe — американская компания, основанная в 1982 году, разработчик программного обеспечения для создания цифрового контента, маркетинга и рекламных технологий. Штаб-квартира расположена в Сан-Хосе (Калифорния). Компания известна продуктами для работы с графикой, видео и веб-дизайном, а также облачной платформой для автоматизации маркетинга и программной закупки рекламы.

## История

Adobe была основана Джоном Уорноком и Чарльзом Гешке. Первоначально компания специализировалась на языке описания страниц PostScript. В 1990-е годы Adobe стала лидером на рынке программного обеспечения для дизайна, выпустив такие продукты, как Photoshop, Illustrator и InDesign. В 2000-х компания начала активно развивать направление маркетинговых технологий, приобретя в 2009 году Omniture (платформа веб-аналитики), что заложило основу для Adobe Experience Cloud. В 2016 году Adobe купила платформу для программной закупки рекламы TubeMogul за $540 млн, что позволило компании выйти на рынок programmatic-рекламы. В 2018 году Adobe завершила сделку по приобретению платформы электронной коммерции Magento за $1,68 млрд. В 2020 году компания приобрела платформу для управления взаимоотношениями с клиентами (CRM) Marketo.

## Роль в рекламной индустрии

Adobe занимает позицию одного из ключевых поставщиков технологий для рекламной индустрии, предлагая решения для программной закупки рекламы (DSP), управления данными (DMP), аналитики и персонализации. Платформа Adobe Advertising Cloud (на базе TubeMogul) позволяет рекламодателям и агентствам покупать медийную, видео- и мобильную рекламу в реальном времени через RTB. Компания также активно развивает инструменты для работы с first-party данными, в частности, продукт Custom Algorithms, который позволяет рекламодателям использовать собственные данные для оптимизации кампаний в DSP. Adobe является одним из участников рынка adtech, конкурируя с такими платформами, как Google (DV360), Amazon (Amazon Ads) и The Trade Desk.

## Ключевые продукты

- **Adobe Advertising Cloud (DSP)** — платформа для программной закупки рекламы (display, video, audio, connected TV), работающая через RTB. Включает инструменты для управления ставками, таргетинга и оптимизации на основе first-party данных.
- **Adobe Experience Cloud** — набор облачных решений для управления клиентским опытом, включающий:
  - **Adobe Analytics** — платформа для веб- и мобильной аналитики.
  - **Adobe Target** — инструмент для A/B-тестирования и персонализации контента.
  - **Adobe Campaign** — платформа для автоматизации маркетинга и управления кампаниями.
- **Adobe Experience Platform** — платформа для управления данными (CDP), позволяющая объединять данные из различных источников для построения единого профиля клиента.
