## Что это за проект

Профессиональное медиа о **рекламе, рекламных технологиях и маркетинге** для России и СНГ. Гибридная модель:
- **ИИ-агенты** пишут новости, дайджесты, обзоры (объём и скорость);
- **эксперты-люди** пишут колонки, аналитику и флагманские отчёты (глубина).

Аудитория двухуровневая: практики (таргетологи, медиабаеры, аналитики) и управленцы (CMO, CEO). Референсы качества: Digiday, The Drum, Detailed.com. Анти-референс: VC.ru.

Полное позиционирование — в `00_Позиционирование_медиа.md`.

## Макеты сайта (`mockups/`)

`mockups/site-mockup.html` — интерактивный HTML-макет двух ключевых страниц (**главная** + **страница материала**), переключаются в служебной панели вверху. Один самодостаточный файл, светлая/тёмная тема.

### Зафиксированные дизайн-решения (сессия 03.07.2026)

Направление собрано по референсам **Every.to** (спокойная off-white база, serif, воздух) и **The Drum / Marketing Brew** (дисциплина в цвете). Итог:

- **Палитра «Петроль».** Один акцент — тёмный бирюзово-синий `#14636B`. Смягчённые чернила `#26262A` (не чистый чёрный), тёплый off-white фон `#F7F5F0`, волосяные линейки `#E3DED3`.
- **Один акцент, без второго цвета.** Использовался переключатель на 5 палитр (Петроль / Индиго / Хвоя / Слива / Охра) для выбора — выбран Петроль, переключатель убран.
- **Контент текстовый, без картинок.** У новостей на главной нет превью-обложек — только текст (заголовок, лид, мета). Картинка допустима **опционально только для главного материала (feature)**, сейчас отключена.
- **Serif-forward типографика.** Заголовки, лиды, тело и заголовки карточек — serif; sans только для навигации, мет и служебных подписей. В макете системные аналоги (Georgia + system-sans); на проде — Source Serif Pro + Inter из ТЗ 02.
- **Категории без цвета.** Раздел 7 ТЗ 02 предлагал цветовую кодировку категорий по гео (4 цвета) — **отклонено**: слишком пёстро. Категории — типографские подписи капсом, гео — маленький нейтральный тег `РФ / МИР / АЗИЯ`.
- **Структура главной:** hero (текстовый) + «Главное сейчас» → две колонки: лента «Свежее» + правый сайд-бар («Дайджест недели», «Популярное» с рейтингом 1–5) → «Мнения» → подписка → футер.
- **Структура материала:** хлебные крошки → категория → H1 → лид (serif italic) → мета с автором → тело (колонка 720px) со спец-блоками (callout, statblock, сравнительная таблица, источник, теги) → карточка автора → «Читайте также» → подписка.

> ⚠️ Отклонения от ТЗ 02, зафиксированные выше (акцентный цвет `#14636B` вместо red-orange `#C8472B`; отсутствие цветокодировки категорий; текст без превью-картинок) — приоритетнее исходного ТЗ. При обновлении `02_Структура_и_дизайн_сайта.md` перенести туда.
