## Что это за проект

Профессиональное медиа о **рекламе, рекламных технологиях и маркетинге** для России и СНГ. Гибридная модель:
- **ИИ-агенты** пишут новости, дайджесты, обзоры (объём и скорость);
- **эксперты-люди** пишут колонки, аналитику и флагманские отчёты (глубина).

Аудитория двухуровневая: практики (таргетологи, медиабаеры, аналитики) и управленцы (CMO, CEO). Референсы качества: Digiday, The Drum, Detailed.com. Анти-референс: VC.ru.

Полное позиционирование — в `00_Позиционирование_медиа.md`.

## Макеты сайта (`mockups/`)

`mockups/1screen-mockup-minimal.html` — **актуальный, канонический макет.** Направление «тех/продукт минимал»: гротеск (sans), чистый белый фон, много воздуха, контент text-only. Интерактивный HTML-макет двух ключевых страниц (**главная** + **страница материала**), переключаются в служебной панели вверху. Самодостаточный файл, светлая/тёмная тема, брендирован под `1screen.ru`, тексты по принципу «анти-ИИ-голос». **Эту вёрстку и стиль сохраняем** — на них строится `media-site` (Итерация 1).

`mockups/1screen-mockup.html` — ранняя «газетная» версия (serif-forward, тёплая off-white «бумага», плотные линейки, буквица). Оставлена как история: направление **отклонено** — читалось как broadsheet-газета (Ведомости), а не как медиа о рекламе. Ориентироваться на минимал.

`mockups/site-mockup.html` — самая ранняя версия с рабочим названием «Охват». История; брендинг и часть текстов устарели.

### Зафиксированные дизайн-решения (сессия 03.07.2026, направление сменено 05.07.2026)

Направление — «тех/продукт минимал»: чистая продуктовая эстетика по референсу **Detailed.com** (sans, белый фон, воздух, текст-форвард) с дисциплиной в цвете от трейд-пресс (**Digiday / The Drum**). Сознательно ушли от газетного broadsheet-сигнала (serif + тёплая «бумага»), который делал сайт похожим на Ведомости. Итог:

- **Палитра «Петроль» на холодном белом.** Один акцент — бирюзово-синий `#14636B` (в макете чуть ярче `#0F6F78` для белого фона, `#0A5A62` для текстовых ссылок — контраст). Нейтралы холодные: чистый белый фон `#FFFFFF`, поднятые поверхности `#F5F7F8`, чернила `#10141B`, линейки `#E7EAEE`. Тёмная тема — почти-чёрный `#0B0D11` + яркий бирюзовый акцент. (Тёплая «бумага» газетного варианта отклонена — читалась как newsprint.)
- **Один акцент, без второго цвета.** Использовался переключатель на 5 палитр (Петроль / Индиго / Хвоя / Слива / Охра) для выбора — выбран Петроль, переключатель убран.
- **Контент текстовый, без картинок.** У новостей на главной нет превью-обложек — только текст (заголовок, лид, мета). В продуктовом минимале это осознанная позиция («не разбавляем стоками»), а не «лента информагентства». Картинка допустима **опционально только для главного материала (feature)**, сейчас отключена.
- **Sans-forward типографика (гротеск).** Заголовки, лиды, тело, подписи — всё sans; serif не используется. Крупные веса (750–800) и плотный трекинг в заголовках; mono (`ui-monospace`) — в мете, датах, цифрах, тегах (продуктовая текстура). В макете системный sans-стек как стенд-ин; на проде — **Inter** (или близкий гротеск).
- **Категории без цвета.** Раздел 7 ТЗ 02 предлагал цветовую кодировку категорий по гео (4 цвета) — **отклонено**: слишком пёстро. Категории — типографские подписи капсом, гео — маленький нейтральный тег `РФ / МИР / АЗИЯ`.
- **Логотип-локап.** Sans-вордмарк `1screen` (цифра «1» — акцентным Петролем) + `.ru` мелкой мéтой, рядом через тонкую линейку дескриптор `Медиа о рекламе и маркетинге` капсом. Дескриптор скрывается в тесной зоне (901–1120px) и на очень узких экранах; вордмарк — всегда. (В газетном варианте вордмарк был serif — теперь sans.)
- **Продуктовые детали.** Акцентные тик-маркеры перед подписями секций (ритм бренд-цвета вместо картинок); hover-стрелка «→» у новостей; залипающий правый сайд-бар; анимированное подчёркивание навигации; тень шапки при скролле; прогресс-бар чтения на материале; пульс «live»-точки. Сдержанно, с учётом `prefers-reduced-motion`.
- **Тексты — по принципу «анти-ИИ-голос».** Вся копия в макете вычищена от ИИ-клише (конструкция «не X, а Y», слово «вердикт», трансформационные зачины «перестало быть… стало», «X ради X»). Оговорки и хеджирование («по данным самого…», «пока рано судить») — сохраняем. Канон — раздел 7 в `00_Позиционирование_медиа.md`.
- **Структура главной:** hero (текстовый) + «Главное сейчас» → две колонки: лента «Свежее» + залипающий правый сайд-бар («Дайджест недели», «Популярное» с рейтингом 1–5) → «Мнения» → подписка → футер.
- **Структура материала:** хлебные крошки → категория → H1 → лид (крупный sans, без буквицы) → мета с автором → тело (колонка ~680px) со спец-блоками (callout, statblock, сравнительная таблица, источник, теги) → карточка автора → «Читайте также» → подписка. Сверху — прогресс-бар чтения.

> ⚠️ Отклонения от ТЗ 02 (приоритетнее исходного ТЗ): акцент `#14636B` вместо red-orange `#C8472B`; **sans-forward вместо serif** (ТЗ 02 предлагал Source Serif Pro для заголовков/тела — отклонено, всё на гротеске Inter); **холодный белый фон вместо тёплого off-white**; нет цветокодировки категорий; контент без превью-картинок. При обновлении `02_Структура_и_дизайн_сайта.md` перенести туда (краткая пометка уже добавлена в шапку ТЗ 02).

## Статус запуска (сессия 06.07.2026)

Этап 0 закрыт, начата Итерация 1 — инфраструктура и главная страница подняты в продакшн.

**Домен.** `1screen.ru` зарегистрирован на **reg.ru**. История проверена (Wayback + `site:` в поиске): в 2018 стоял заброшенный дефолтный Drupal, спам-прошлого нет — чистый. DNS-серверы reg.ru (`ns1.reg.ru` / `ns2.reg.ru`).

**Сайт (`media-site/`).** Стек **Astro 5 + Tailwind 4 + Inter** (self-hosted через Fontsource `@fontsource-variable/inter`, без CDN). Собрана **главная страница** — перенос канонического макета `mockups/1screen-mockup-minimal.html` один в один (палитра «Петроль», sans, text-only). Контент на главной пока демонстрационный.

**Деплой (Vercel).** Team `1screen` (Hobby, бесплатно), проект `media`, **Root Directory = `media-site`**, автодеплой из ветки `main`. Публичный сайт: `https://1screen.ru` (авто-SSL). `www.1screen.ru` → **308 permanent redirect** на apex. **Deployment Protection = Standard Protection** (публичный домен открыт всем, служебные `*.vercel.app` под входом — это норма, не трогать).
